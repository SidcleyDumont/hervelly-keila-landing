## 🚀 Deploy no Vercel

1. Acesse [vercel.com/new](https://vercel.com/new)
2. Importe este repositório do GitHub
3. Clique em **Deploy** (não precisa configurar nada — é HTML estático)
4. Sua URL pública será gerada automaticamente

### Domínio próprio (opcional)

Após o deploy, em **Settings → Domains**, adicione seu domínio (ex: `hervellykeila.com.br`) e siga as instruções de DNS.
